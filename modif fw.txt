2. Dans ton fork device-ui
Fichier à modifier
include/graphics/driver/LGFXDriver.h

C’est le vrai fichier à patcher.

Code exact à modifier dans LGFXDriver.h
A. Ajouter Wire.h

En haut du fichier, avec les autres includes, ajoute :

#include <Wire.h>

Par exemple, tu auras quelque chose comme :

#include "LovyanGFX.h"
#include "graphics/driver/DisplayDriverConfig.h"
#include "graphics/driver/TFTDriver.h"
#include "input/InputDriver.h"
#include "lvgl_private.h"
#include "util/ILog.h"
#include <functional>
#include <Wire.h>
B. Ajouter le helper T-Deck

Dans le même fichier, après les constexpr du haut :

constexpr uint32_t defaultLongPressTime = 600;
constexpr uint32_t defaultGestureLimit = 10;
constexpr uint32_t defaultScreenTimeout = 30 * 1000;
constexpr uint32_t defaultBrightness = 153;

ajoute ce bloc :

#ifdef T_DECK
static constexpr uint8_t TDECK_KB_I2C_ADDR = 0x55;
static constexpr uint8_t TDECK_KB_BRIGHTNESS_CMD = 0x01;
static constexpr uint8_t TDECK_KB_WAKE_BRIGHTNESS = 127;

static inline void tdeckKeyboardBacklight(uint8_t level)
{
    Wire.beginTransmission(TDECK_KB_I2C_ADDR);
    Wire.write(TDECK_KB_BRIGHTNESS_CMD);
    Wire.write(level);
    Wire.endTransmission();
}
#endif
C. Couper le rétroéclairage clavier à l’entrée en veille

Dans la fonction :

template <class LGFX> void LGFXDriver<LGFX>::task_handler(void)

repère ce bloc :

ILOG_INFO("enter powersave");
DisplayDriver::view->screenSaving(true);
if (hasTouch() && hasButton()) {
    ILOG_DEBUG("disable touch, enable button input");
    lv_indev_enable(DisplayDriver::touch, false);
    lv_indev_enable(InputDriver::instance()->getButton(), true);
}
lgfx->sleep();
lgfx->powerSaveOn();
powerSaving = true;

et remplace-le par :

ILOG_INFO("enter powersave");
DisplayDriver::view->screenSaving(true);
#ifdef T_DECK
tdeckKeyboardBacklight(0);
#endif
if (hasTouch() && hasButton()) {
    ILOG_DEBUG("disable touch, enable button input");
    lv_indev_enable(DisplayDriver::touch, false);
    lv_indev_enable(InputDriver::instance()->getButton(), true);
}
lgfx->sleep();
lgfx->powerSaveOn();
powerSaving = true;
D. Rallumer le rétroéclairage clavier au réveil

Toujours dans task_handler(), repère ce bloc :

ILOG_INFO("leaving powersave");
powerSaving = false;
DisplayDriver::view->triggerHeartbeat();
lgfx->powerSaveOff();
lgfx->wakeup();
lgfx->setBrightness(lastBrightness);
DisplayDriver::view->screenSaving(false);

et remplace-le par :

ILOG_INFO("leaving powersave");
powerSaving = false;
DisplayDriver::view->triggerHeartbeat();
lgfx->powerSaveOff();
lgfx->wakeup();
lgfx->setBrightness(lastBrightness);
#ifdef T_DECK
tdeckKeyboardBacklight(TDECK_KB_WAKE_BRIGHTNESS);
#endif
DisplayDriver::view->screenSaving(false);
Ce que fait exactement cette modif
quand l’écran entre en powersave :
tdeckKeyboardBacklight(0);
quand l’écran se réveille :
tdeckKeyboardBacklight(127);

Ça correspond exactement à ton besoin, et c’est la modif qui a fonctionné chez toi.

Ce que tu peux ignorer / retirer

Les modifs faites auparavant dans :

src/input/kbI2cBase.h
src/input/kbI2cBase.cpp
src/graphics/Screen.cpp

ne sont pas nécessaires pour t-deck-tft.
Tu peux les revert dans ton dépôt firmware si tu veux garder quelque chose de propre.

Workflow propre final
Dans le fork device-ui
modifier include/graphics/driver/LGFXDriver.h
commit
push
Dans le dépôt firmware
modifier variants/esp32s3/t-deck/platformio.ini
remplacer device-ui par ton fork dans lib_deps
rebuild
Build

Dans WSL :

cd ~/fw
source ~/.venvs/pio/bin/activate
rm -rf .pio
platformio run -e t-deck-tft
